# ParaUtils


## Installation

### Maven

The following is the installation guide for Maven.

**Repository:**
```
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>
```

**Dependency:**
```
<dependency>
    <groupId>com.github.paralian-inc</groupId>
    <artifactId>ParaUtils</artifactId>
    <version>v0.1</version>
</dependency>
```
